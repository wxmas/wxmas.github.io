1. 运行 Jekyll：
`~$ bundle exec jekyll server`

2. 将博文以markdown格式文件，保存在`_posts/`目录下

3. 将仓库内容push到网站上

	`~$ git add --all`
	
	`~$ git commit -m "Initial commit"`

	`~$ git push -u origin master`
